================================================================================
TECHNICAL ARCHITECTURE
================================================================================

Decision Engine (server/decision-engine/):
- Layer 1: types.ts - Canonical type definitions
- Layer 2: questionMapper.ts - Question to signal extraction
- Layer 3: dimensionMapper.ts - Signal to dimension promotion
- Layer 4: decisionTable.ts - Verdict logic with strict precedence
- Layer 5: effortAllocator.ts - Verdict to effort allocation
- Layer 6: contradictionDetector.ts - Soft flags
- Layer 7: engine.ts - Orchestrator

Key Design Principles:
- Pure rule-based system (NO scoring math, NO ML, NO AI in verdict logic)
- Numeric scores retained for PDF display only
- GPT-4.1 generates narrative explanations only
